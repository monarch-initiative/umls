# umls-ingest

UMLS ingest to generate mappings.

# Setup

```
git clone https://github.com/monarch-initiative/umls-ingest.git
cd umls-ingest
make mappings
```
This should kickstart the process of an SSSOM tsv generation named `umls.sssom.tsv`.
